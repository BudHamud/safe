# üè¶ CajaFuerte - Financial Management System V2

**CajaFuerte** (Strongbox) is a comprehensive expense and income management application designed with a premium, minimalist, and functional aesthetic. It provides total control over your finances with real-time multi-currency support and an optimized user experience for both web and mobile devices (Android).

## üìä Key Features

- **Multi-Currency Management:** Native support for **ILS, USD, ARS, and EUR**. Automatic conversion and balance visualization in your preferred base currency with live exchange rates.
- **Category Discovery:** An intelligent system that identifies your categories from transaction history, featuring custom icon support and total personalization.
- **Travel Mode / New Account:** Isolate specific movements and balances for trips or specific periods without affecting your main historical data.
- **Premium Interface:** Dark mode design inspired by modern trends, featuring *glassmorphism* effects, micro-animations, and carefully curated typography (Outfit & Inter).
- **Data Security:** Card digit masking for credit cards and secure transaction handling.
- **Statistical Dashboard:** Clear visualization of fixed vs. variable expenses, monthly goal tracking, and total balance overview.
- **Multi-Platform:** Works as a Web application (Next.js) and as a Native Android app (Capacitor).

## üöÄ Local Setup

1. **Clone the repository and install dependencies:**
   ```bash
   npm install
   ```

2. **Configure environment variables:**
   Create a `.env` file in the root directory with your PostgreSQL URL:
   ```env
   DATABASE_URL="postgresql://user:password@host:port/dbname"
   ```

3. **Sync the database schema:**
   ```bash
   npx prisma db push
   ```

4. **Run in development mode:**
   ```bash
   npm run dev
   ```

## üåê Deployment

### 1. Database (Railway)
- Create a project on Railway and add a PostgreSQL service.
- Use the `DATABASE_PUBLIC_URL` for the initial synchronization (`db push`).

### 2. Web (Vercel)
- Connect your GitHub repository to Vercel.
- Add the `DATABASE_URL` environment variable.
- The build command will automatically run `prisma generate` due to the `postinstall` script.

### 3. Mobile (Android)
- Update the server URL in `capacitor.config.ts` to your Vercel production URL.
- Execute:
  ```bash
  npm run build
  npx cap sync android
  npx cap open android
  ```
- In Android Studio, generate your APK from the *Build* menu.
